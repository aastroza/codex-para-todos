# Codex para todos

Landing page de la comunidad **Codex para todos**, creada para compartir recursos, encuentros y formas simples de aprender a trabajar con Codex sin ser programador.

## Desarrollo

```bash
npm install
npm run dev
```

El sitio queda disponible en `http://127.0.0.1:4321/`.

## Build y despliegue

```bash
npm run build
npm run deploy
```

El proyecto está construido con Astro y se despliega en Cloudflare Pages.
